# Animated Countdown

A sleek, dynamic animated countdown timer built with HTML, CSS, and Vanilla JavaScript. This project visually counts down from 10 to 0 with smooth entrance/exit animations before displaying a final "GO" message and an option to replay the animation.

## Features

- **Smooth CSS Animations:** Numbers animate in and out seamlessly using CSS keyframes.
- **Dynamic JavaScript Logic:** Handles the timing, DOM event listeners for animations, and resetting the countdown. 
- **Responsive & Clean UI:** Simple, centered interface that looks great across devices.

## How to Run

1. Clone or download this repository.
2. Open `index.html` in your favorite web browser.
3. No build tools or dependencies required!

## Project Structure

- `index.html`: The main structural layout of the countdown interface.
- `style.css`: Contains all styles, layouts, and keyframe animations for the number transitions.
- `script.js`: Provides the interactivity, adding/removing animation classes to cycle through the numbers.
